# finance-dashboard
# 🇮🇳 Indian Personal Finance Dashboard

An interactive data visualization dashboard built with Python, Streamlit and Plotly
to explore spending habits, savings patterns and financial behaviour 
across 20,000 individuals in India.

## Features
- Filter by City Tier, Occupation, Age Group, Dependents and Income Range
- KPI cards for Income, Expenses, Disposable Income and Savings
- 8 interactive charts covering spending breakdown, potential savings,
  savings gap, age trends and more

## Tech Stack
- Python
- Streamlit
- Plotly
- Pandas

## How to run locally
pip install -r requirements.txt
streamlit run dashboard.py

## Dataset
20,000 records with income, expense categories, desired savings
and potential savings across Tier 1, 2 and 3 Indian cities.
